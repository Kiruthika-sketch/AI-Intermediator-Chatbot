🤖 AI Intermediator Chatbot

A simple AI-powered chatbot built using FastAPI for the backend and a basic frontend using HTML, CSS, and JavaScript. It can process user queries and return intelligent responses using an AI model.

🚀 Features
Chat interface for user interaction
FastAPI backend for handling requests
AI-based response generation
Simple frontend (HTML/CSS/JS)
Easy to extend and improve
🛠️ Tech Stack
Python
FastAPI
HTML, CSS, JavaScript
(Optional) OpenAI / LLM integration

▶️ How to Run
1. Clone the repo
git clone https://github.com/Kiruthika-sketch/AI-Intermediator-Chatbot.git
cd AI-Intermediator-Chatbot
2. Run Backend
cd backend
uvicorn main:app --reload
3. Run Frontend
cd frontend
python -m http.server 8000

Then open:

http://localhost:8000
